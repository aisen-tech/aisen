기술

시스템 구성

우리 플랫폼은 프론트엔드, AI 분석 엔진, 데이터 관리 계층으로 구성되어 있으며, 각 요소는 확장성과 신뢰성을 고려하여 설계되었습니다.

프론트엔드

Next.js와 React 기반의 웹 인터페이스를 통해 사용자는 손쉽게 영상 업로드, 분석 결과 확인, 대시보드 조회 등을 수행할 수 있습니다.

AI 분석 엔진

YOLOv8, OpenPose, 얼굴 인식 모델을 통합하여 안전 장비 착용 여부, 작업 자세, 무단 침입 등을 감지하며 GPU와 서버리스 환경에서 효율적으로 동작합니다.

데이터 & 인프라

원본 영상과 분석 결과는 암호화된 상태로 S3에 저장되고, 감지 이벤트와 메타데이터는 DynamoDB에 기록되며 AWS Lambda와 Fargate를 통해 확장성 있는 처리를 지원합니다.

사용 기술

AI 기반 영상 분석 플랫폼은 여러 계층의 기술로 구성되어 있으며, 최신 웹 개발과 클라우드 인프라를 활용해 높은 확장성과 안정성을 제공합니다.

프론트엔드

  • Next.js 15과 React 19 기반의 SPA/SSR 구조
  • TypeScript 및 Tailwind CSS와 shadcn/ui 컴포넌트로 일관된 UI 제공
  • CloudFront를 통해 전 세계 어디서나 빠른 로딩 속도 보장

백엔드 & 인프라

  • AWS API Gateway와 Lambda로 구성된 서버리스 API 계층
  • 영상 길이에 따라 Lambda, ECS Fargate, GPU 인스턴스를 조합한 유연한 처리
  • SQS 큐로 작업을 분산 관리하고 자동 재시도를 지원
  • DynamoDB와 S3에 메타데이터와 영상 데이터를 암호화하여 저장

AI/ML

  • YOLOv8로 안전장비 착용 여부와 작업자·중장비를 감지
  • OpenPose를 통해 작업자의 자세를 분석하고 위험 동작을 분류
  • FaceNet/dlib 기반 얼굴 인식으로 무단 침입자 식별
  • OpenCV와 SageMaker를 활용해 영상 전처리와 모델 호스팅을 최적화

보안 & 운영

  • 저장·전송 구간 모두 암호화하여 데이터 보호
  • 정기 백업과 버전 관리로 데이터 손실 위험 최소화
  • AWS CDK와 Docker로 인프라를 코드로 관리하여 지속적인 배포 자동화
  • CloudWatch와 SNS를 통해 서비스 상태를 모니터링하고 알림을 제공합니다
